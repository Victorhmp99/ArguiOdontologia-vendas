# Argui Odontologia — Landing Page de Vendas

Landing page de conversão para a Argui Odontologia e Estética Orofacial, com foco em implantes e próteses dentárias, na Asa Sul, Brasília.

## Stack
HTML + CSS + JS puro, sem build. Basta abrir `index.html` no navegador ou servir a pasta com qualquer servidor estático.

## Estrutura
```
index.html
css/style.css
js/main.js
assets/img/   <- fotos da clínica, da equipe e casos de antes/depois
```

## Deploy
Compatível com GitHub Pages, Vercel, Netlify, Hostinger ou Hostgator (deploy estático simples).
